### Learnify (EduLearn) — Online Learning Management System
**Status:** Live | **Duration:** March 2026 – Present

<img align="right" width="280" src="https://user-images.githubusercontent.com/74038190/212257467-871d32b7-e401-42e8-a166-fcfd7baa4c6b.gif" alt="Microservices" />

- **7 Microservices:** Auth, Course, Content, Enrollment, Assessment, Progress, Review
- **API Gateway:** YARP | **Frontend:** React dashboards + Angular admin panel
- **Deployment:** Docker containers on Azure

```
Backend:  .NET 8 | ASP.NET Core | EF Core 8 | SQL Server
Frontend: React.js | Angular | HTML5 | CSS3 | JavaScript
Cloud:    Azure Blob | RabbitMQ | MassTransit | Redis
Auth:     JWT | Google OAuth 2.0 | PBKDF2
```

- PBKDF2 hashing + JWT (24h expiry) + Google OAuth 2.0
- QuestPDF certificate generation stored on Azure Blob
- Redis caching (5-min TTL) | MassTransit + RabbitMQ event-driven architecture

[Backend API](https://github.com/ojas2005/Learnify) | [Frontend](https://github.com/ojas2005/LearnifyFrontend) | [Live Demo](https://learnify-frontend-ten-sable.vercel.app)
