### 📚 **Learnify (EduLearn) - Online Learning Management System**
**Status:** 🟢 Live | **Duration:** March 2026 - Present

**Architecture:**
- **7 Microservices:** Auth, Course, Content, Enrollment, Assessment, Progress, Review
- **API Gateway:** YARP (Yet Another Reverse Proxy)
- **Frontend:** React dashboards + Angular admin panel
- **Deployment:** Docker containers on Azure

**Tech Stack:**
```
Backend:  .NET 8 | ASP.NET Core | EF Core 8 | SQL Server
Frontend: React.js | Angular | HTML5 | CSS3 | JavaScript
Cloud:    Azure Blob | RabbitMQ | MassTransit | Redis
Auth:     JWT | Google OAuth 2.0 | PBKDF2
```

**Key Features:**
- 🔐 **Secure Authentication:** PBKDF2 hashing + JWT tokens (24h expiry) + Google OAuth 2.0
- 📜 **Certificate Generation:** QuestPDF integration for automated certificates stored on Azure Blob
- ⚡ **Performance:** Redis caching with 5-minute TTL for frequently accessed data
- 📨 **Async Processing:** MassTransit + RabbitMQ for event-driven architecture
- 📊 **Responsive UIs:** Separate dashboards for students and admin
